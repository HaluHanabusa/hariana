# Constraint Matrix(制約行列)

- 行 = アイデアの選択肢 / 列 = 満たすべき条件(Phase 0で決めた制約)
- マスの値 = PASS / FAIL / UNKNOWN。条件や根拠は脚注(a)(b)…で下に書く
  (値+脚注の二層で記録する。マスに長文を書かない)
- 行には状態がある: 正規 / 候補(証拠不足で仮置き) / REJECTED保持
  (却下されても行は消さない。蘇生条件と履歴が資産になる)
- 相反する強みを持つ行(どちらも捨てられない)は、無理に1行にせず分割して両方残す
