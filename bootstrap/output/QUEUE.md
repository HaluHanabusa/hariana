# Queue(タスクの待ち行列)

書式: `- [状態] (Skill or 役割 / 推奨モデル) やること → 出力先`
状態: TODO(未着手) / DOING(作業中、常に1件まで) / DONE(完了) / BLOCKED(人間の判断待ち)

ルール:
- 上から順に処理する
- 1タスクは「1回の作業で終わる大きさ」に割る(目安: Claim 5件以内)
- 人間はいつでもTODOを追記してよい。AIは並び替えをしない
- BLOCKEDにしたら、理由と質問を QUESTIONS.md に書いて次のタスクへ進む
