## 全40原理

1. **分割** — 対象を独立した部分に分ける。分解・組立を容易にする。
   例: コンテナ輸送(貨物の標準単位化) / マイクロサービスアーキテクチャ
2. **分離・抽出** — 邪魔な部分だけを取り除く、必要な部分だけを取り出す。
   例: エアコンの室外機分離(騒音源の隔離) / 音源分離(ボーカル抽出)
3. **局所的性質** — 均一構造をやめ、場所ごとに最適な性質を与える。
   例: 消しゴム付き鉛筆 / 傾斜機能材料(表面だけ硬く内部は靭性)
4. **非対称** — 対称な形を非対称にする、非対称性を強める。
   例: 人間工学マウス / 車のすれ違い用ヘッドライト配光
5. **併合** — 同種・隣接の操作や物体を空間的・時間的に結合する。
   例: マルチコアCPU / 連節バス
6. **汎用性** — 一つの物に複数の機能を持たせ、他の部品を不要にする。
   例: スマートフォン(電話+カメラ+財布) / ソファベッド
7. **入れ子** — 物体を別の物体の中に順に収める。空洞を通す。
   例: テレスコピックアンテナ / 入れ子式計量カップ
8. **つり合い** — 浮力・揚力・他の物体との結合で重量を相殺する。
   例: クレーンのカウンターウェイト / 熱気球
9. **先取り反作用** — 後で生じる応力に備え、事前に逆の応力を与えておく。
   例: プレストレストコンクリート / 予防接種
10. **先取り作用** — 必要な変化・作用の全部か一部を事前に済ませておく。
    例: 糊付き封筒 / CPUのプリフェッチ・投機実行
11. **事前保護** — 信頼性の低い対象に、あらかじめ補償手段を用意する。
    例: エアバッグ / データの自動バックアップ
12. **等ポテンシャル** — 持ち上げ・降ろしが不要になるよう条件を変える。
    例: 同一高さのローラーコンベア搬送 / バリアフリーのフラット動線
13. **逆発想** — 逆の作用にする。動く物を固定し、固定物を動かす。上下を反転。
    例: ランニングマシン(人でなく床を動かす) / 回転寿司(客でなく皿が回る)
14. **曲面化** — 直線・平面を曲線・曲面に。ローラー・ボール・らせんを使う。
    例: アーチ橋(曲げを圧縮に変換) / ボールペン
15. **ダイナミック性** — 対象や環境を、動作条件ごとに最適になるよう可変にする。
    例: 航空機の可変翼・フラップ / 環境光で変わる自動輝度ディスプレイ
16. **部分的・過剰作用** — 100%ちょうどが難しいなら、少し多め/少なめにして後で調整。
    例: 鋳造で過剰に盛って後加工 / 概算で先に引き当てて後で精算する在庫システム
17. **他次元への移行** — 1次元を2次元・3次元に。多層化する。裏面も使う。
    例: 立体駐車場 / 3D NANDフラッシュ(平面微細化の限界を積層で回避)
18. **機械的振動** — 振動させる。周波数を上げる。超音波を使う。
    例: 超音波洗浄 / コンクリートのバイブレータ充填
19. **周期的作用** — 連続作用を周期的・パルス的にする。休止を利用する。
    例: PWM制御(パルスで実効電力を調整) / 点滅サイレン(視認性向上)
20. **有用作用の継続** — 全部分を常に全力で働かせる。アイドルをなくす。
    例: 往復両方向で印字するプリンタヘッド / アイドル時間に走るバックグラウンド処理
21. **高速実行** — 有害・危険な工程を高速で駆け抜ける。
    例: 高速切断による熱変形の回避 / 食品の瞬間冷凍(細胞破壊の抑制)
22. **災い転じて福となす** — 有害な要因・作用を利用して有用な効果を得る。
    例: 回生ブレーキ(制動熱を電力に) / 廃熱利用のコージェネレーション
23. **フィードバック** — フィードバックを導入する。既にあるなら強化・反転する。
    例: サーモスタット / A/Bテストによる継続的改善
24. **仲介** — 中間媒体・仲介物を使う。一時的に別の物体と結合させる。
    例: 触媒 / APIゲートウェイ・アダプタ層
25. **セルフサービス** — 対象自身に補助機能を果たさせる。廃棄物・余剰を自家利用。
    例: 自己修復材料 / セルフレジ
26. **代替・コピー** — 高価・壊れやすい物を、安価なコピー・光学像で置き換える。
    例: フライトシミュレータ訓練 / 衛星画像による現地調査の代替
27. **安価な短寿命** — 高価な長寿命品を、安価な使い捨て品の集まりで置き換える。
    例: 使い捨て医療器具(滅菌コストの回避) / 紙コップ
28. **機械システムの代替** — 機械的手段を光・音・熱・電磁場に置き換える。
    例: 非接触ICカード(物理接点の排除) / 光学式マウス(ボールの排除)
29. **空気圧・水圧** — 固体部品の代わりに気体・液体を使う。
    例: エアクッション梱包 / 油圧ブレーキ
30. **柔軟な殻と薄膜** — 立体構造の代わりに柔軟な膜・薄膜を使う。隔離に使う。
    例: 膜構造建築(東京ドーム型) / 食品ラップによる鮮度保持
31. **多孔質材料** — 物体を多孔質にする。孔に機能物質を仕込む。
    例: 活性炭フィルタ / 発泡金属(軽量+衝撃吸収)
32. **色の変化** — 色・透明度を変える。見えないものを可視化する。
    例: 感温インク(温度の可視化) / 偏光サングラス
33. **均質性** — 相互作用する物体を同じ材料(または近い性質)で作る。
    例: 同種金属同士の溶接(異種接合の腐食回避) / ダイヤモンド研磨にダイヤ粉
34. **排除と再生** — 役目を終えた部分をその場で消す・分離する。消耗部をその場で再生。
    例: 体内で溶けるカプセル・縫合糸 / ロケットの多段切り離し
35. **パラメータ変更** — 濃度・密度・柔軟性・温度・相などの物性を変える。
    例: 液体石鹸(固体→液体で計量性向上) / 低温物流(温度パラメータで鮮度を制御)
36. **相変化** — 相転移に伴う体積変化・潜熱の吸放出を利用する。
    例: ヒートパイプ(蒸発・凝縮で熱輸送) / 蓄熱材PCM(潜熱で温度を平準化)
37. **熱膨張** — 材料の熱膨張・収縮を利用する。膨張係数の異なる材料を組み合わせる。
    例: バイメタルサーモスタット / 焼きばめ(加熱膨張で圧入)
38. **強い酸化剤** — 空気を酸素富化空気に、さらに純酸素に置き換える。
    例: 酸素富化燃焼(燃焼効率向上) / オゾン殺菌
39. **不活性環境** — 通常環境を不活性雰囲気・真空に置き換える。
    例: 窒素充填包装(酸化防止) / アルゴンアーク溶接
40. **複合材料** — 均質材料を複合材料に置き換える。
    例: CFRP(軽量+高剛性) / 鉄筋コンクリート(圧縮に強い材+引張に強い材)
