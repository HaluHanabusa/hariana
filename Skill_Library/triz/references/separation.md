# 分離原理(物理矛盾の解消)

物理矛盾 = 同じ対象に「Aであれ」と「Aでないであれ」を同時要求している状態。
以下の4軸で「同時」を解体できないか順に検討する:

1. **時間による分離** — 別の時刻ならAとnot Aは両立する
   (例: 表示していない瞬間だけ光源を止める → デューティ駆動)
2. **空間による分離** — 別の場所ならAとnot Aは両立する
   (例: 視線中心だけ高輝度、周辺は低輝度)
3. **条件による分離** — 条件が変われば両立する
   (例: 屋外モードと屋内モードで輝度要求を切り替える)
4. **システムレベルによる分離** — 部分と全体で性質を変える
   (例: グラス単体は非力、パックと合わせたシステム全体では高性能)
