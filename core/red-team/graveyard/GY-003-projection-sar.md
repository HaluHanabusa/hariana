# Lightform / RoomAlive系 商用SAR
学術実証(RoomAlive)から11年間商用化に至らず、商用挑戦者(Lightform)は
撤退、後継製品は火災リコール。死因は複合的だが、シーン理解レイテンシの
定量的空白と設置コスト転嫁が主因の時期的死と分類。
蘇生条件: シーン理解プロジェクションの遅延に関する定量データの空白が解消されること、
単一ユーザー・限定環境への要求緩和。
出典: 公知の事実(RoomAlive発表2014、Lightformの撤退、後継製品のリコール)に基づく
